Prompt for Replit agent

I want to standardize the site header and remove the image logo from home and demo.
Please propose the plan below for my approval, then execute it.

Goals
	•	Replace the current “horrible logo” on / and /demo with a clean text wordmark: skyla.dev.
	•	Use one shared header (same markup, styles, and behavior) across /, /demo, /docs, and /sign-in.
	•	Keep all existing page content and routes working.

Plan
	1.	Create a shared header partial and a tiny injector script:
	•	/public/partials/header.html – markup for a minimal, consistent header (no image; just text wordmark + right-aligned nav).
	•	/public/header.js – finds <div id="site-header"> and injects the partial via fetch('/partials/header.html').
	2.	Add a global stylesheet used by every page:
	•	/public/global.css – header layout, typography, mobile tweaks; neutral to each page.
	3.	Wire every page to the shared header:
	•	Add <link rel="stylesheet" href="/global.css"> in <head>.
	•	Add <div id="site-header"></div> at the top of <body>.
	•	Add <script src="/header.js" defer></script> before </body>.
	•	Remove any page-local logo blocks or duplicate nav bars.
	4.	QA:
	•	Verify header matches on /, /demo, /docs, /sign-in.
	•	Confirm the /demo console still works; no layout shifts; no 404s for assets.
	•	Mobile: header wraps nicely, same spacing on all pages.

If this plan looks good, implement with the exact files below.

/public/header.js

(async () => {
  const mount = document.getElementById('site-header');
  if (!mount) return;
  try {
    const res = await fetch('/partials/header.html', { cache: 'no-store' });
    mount.innerHTML = await res.text();
  } catch (e) {
    // Silent fail; page still works without header
    console.error('Header injection failed', e);
  }
})();
